# velbusLib
simple project to exchange between web server and client browser and IP Velbus server

I've created this project to learn myself with NodeJS technologies and to create my own apps with it.
I know well Velbus protocol because I've written PB_Velbus-server (a serial/IP server running on Windows platform) and some personnal others applications :
- Tea (for temperature graph on all sensors)
- Teo (for acting as an alarm when I leave my house)
- Veltron (put graphic symbols of blinds, relays, buttons upper a house plan and to be able for status and action)
- velbuscmd (simplify actions sending)
- Velcheck (to check errors bytes on all modules)

This project is the beginning for others projects but let me play with JavaScript and asynchronous possibilities.
